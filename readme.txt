# Scope India Institute Website Clone (Full Stack Django Project)

 📌 Description

Developed a full-stack web application replicating an educational institute website (Scope India) using Django (Python). The platform showcases courses, pricing, placement details, and includes student registration and dashboard functionalities along with contact and email handling features.

 🚀 Features

* Course listing with detailed descriptions and pricing
* Display of recent placements and institute highlights
* Student registration system
* Contact form with email handling functionality
* Role-based authentication system
* Student dashboard for personalized access
* Responsive and user-friendly UI

## 🛠️ Tech Stack

* **Backend:** Python (Django)
* **Frontend:** HTML, CSS, JavaScript, React.js
* **Database:** MySQL

## ⚙️ Setup Instructions

### 1. Clone the repository

git clone https://github.com/Madhav-A1408/Scope-India-Institute-Website-Clone-Full-Stack-Django-Project.git

### 2. Navigate to the project directory

cd Scope-India-Institute-Website-Clone-Full-Stack-Django-Project

### 3. Create virtual environment (optional but recommended)

python -m venv venv
venv\Scripts\activate

### 4. Install dependencies

pip install -r requirements.txt

### 5. Apply migrations

python manage.py migrate

### 6. Run the server

python manage.py runserver

## 📂 Note

* The database file and media uploads are excluded for security and performance reasons.
* Run migrations to generate the database locally.
* Configure email settings in Django for contact form functionality.
